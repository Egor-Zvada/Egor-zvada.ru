# Changelog

## v09-release

Добавлено и изменено:

- версия сайта обновлена до `v09-release`;
- добавлен финальный SEO-набор в `partials/head.php`: canonical, robots, Open Graph, Twitter Card, author, manifest;
- добавлены `robots.txt`, `sitemap.xml`, `site.webmanifest`;
- добавлен Open Graph SVG-баннер `assets/img/og-image.svg`;
- добавлено мобильное меню с кнопкой открытия/закрытия;
- добавлены accessibility-правки: skip-link, корректные `aria-expanded`, обработка Escape для меню;
- добавлена финальная адаптивная полировка hero, header, секций и мобильных размеров;
- добавлен CSS-блок для `prefers-reduced-motion`;
- обновлён footer build-label до `v09-release`;
- добавлены deploy-примеры для nginx и Apache;
- добавлен `.htaccess` для Apache;
- обновлён `README.md` под релизную версию;
- выполнена проверка PHP/JS-синтаксиса и базовой отдачи страницы.

Что осталось после релиза:

- заменить SVG-заглушки реальными фото/видео проектов;
- при необходимости добавить реальные страницы ошибок в текущей айдентике;
- после заливки проверить сайт на реальном домене и внести точечные правки по верстке;
- затем можно переходить к расширению лендинга в полноценный сайт.

## v08-motion

- усилена большая canvas-анимация на первом экране;
- hero-анимация стала ближе к живой цифровой ткани / pseudo-3D mesh;
- добавлена реакция hero-анимации на курсор и инерцию скролла;
- добавлен общий файл `assets/js/motion-config.js`;
- добавлена вторая визуальная canvas-сцена в блоке «Почему Таврида»;
- обе анимации учитывают dark/light mode и `prefers-reduced-motion`;
- добавлены паузы/ограничения через IntersectionObserver.
